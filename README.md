# Locus Workspace

Canvas workspace for organizing Unity assets and scene references inside Locus.

## What it does

Locus Workspace provides a canvas View where you can drag Unity assets, search project assets, arrange references into pages, and select or inspect those references from Locus.

## How to use

Install the plugin in Locus, then open the **Locus Workspace** View. Drag assets from Unity into the canvas, or right-click/search inside the View to add references.
